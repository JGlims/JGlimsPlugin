## BetterModel Models (.bbmodel files → plugins/BetterModel/models/)

Every custom mob requires a .bbmodel file created in Blockbench. Place in `plugins/BetterModel/models/` on the server and run `/bettermodel reload`.

## Dimension-Specific Notes

### Aether (world_aether)
- Sky texture: bright, heavenly (custom sky texture if possible)
- Block palette: quartz, gold blocks, glowstone, prismarine, cyan terracotta, white concrete, glass

### Lunar (world_lunar)  
- Sky texture: dark with Earth visible
- Block palette: gray concrete, iron blocks, deepslate, blackstone, tinted glass, redstone lamps, end stone

### Jurassic (world_jurassic)
- Uses normal Overworld sky
- Block palette: bone blocks, mossy stone, coarse dirt, mud, terracotta, jungle wood
